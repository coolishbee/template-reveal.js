# template-reveal.js

[reveal.js](https://github.com/hakimel/reveal.js)는 웹 브라우저에서 동작하는 오픈소스 HTML 프레젠테이션 프레임워크입니다. HTML, CSS, JavaScript만으로 슬라이드를 만들 수 있으며, 별도 발표 도구 없이 브라우저에서 바로 실행할 수 있습니다.

## 주요 특징

- 중첩 슬라이드 지원
- 마크다운 기반 슬라이드 작성
- 코드 하이라이팅
- 발표자 노트
- PDF 내보내기
- Auto-Animate 전환 효과
- 플러그인과 API를 통한 확장

## 이 프로젝트에서의 사용

이 저장소는 reveal.js를 이용해 정적 HTML/CSS 기반 발표 자료를 구성합니다. 슬라이드 내용은 `index.html`에 작성되어 있으며, 스타일은 `styles.css`에서 관리합니다.

## 실행 방법

```bash
npm install
npm start
```

실행 후 브라우저에서 `http://localhost:3000`으로 접속합니다.
